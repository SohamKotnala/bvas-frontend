Bill Verification & Audit System (BVAS)

Roles:
- Vendor: Create and submit bills
- District Verifier: Verify district bills
- HQ Admin: Audit, lock/unlock bills

Tech Stack:
- Frontend: React
- Backend: Node.js + Express
- Database: PostgreSQL

Key Features:
- Role-based access control
- Bill lifecycle with rejection & resubmission
- EPOS vs Vendor quantity comparison
- Full audit trail
- HQ override controls
